# SDLSnake
Snake in C with SDL Library (only on Mac & Linux)

## Requirements
<ul>
  <li><a href="https://www.libsdl.org/">SDL2<a/></li>
</ul>
For macOS user, install SDL2 in /Library/Frameworks directory or change to your custom SDL2 path.

## In terminal

### On mac
`make build_mac`

### On Linux
`make build_linux`

### Run

`./main`
